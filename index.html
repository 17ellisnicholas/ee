HTML5 Bombergirl

How to play

Player 1

  • arrow keys … move
  • spacebar … plant the bomb

Player 2

  • W, A, S, D … move
  • shift … plant the bomb
M … mute / unmute sound
enter … restart
This game has nothing to hide. Download the source code on GitHub.
